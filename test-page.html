📖 Testing Instructions

  • Click the extension icon and click "Start Picking"
  • Hover over elements to see them highlighted in blue
  • Click any element to inspect it
  • Open the extension popup to see the element's:
    • Tag name
    • Classes (including parent classes)
    • ID (if present)
    • Unique CSS selector
  • Use the copy buttons to copy the selector or classes
  • Press ESC to stop picking mode
