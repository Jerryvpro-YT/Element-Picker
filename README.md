# 🎯 Element Picker - Chrome Extension

Pick any element on a webpage and instantly copy its CSS selector or classes!

## ✨ Features

- **Click & Inspect:** Click any element on a page to inspect it.
- **Instant Copy:** Copy the CSS selector or all class names with one click.
- **Clean UI:** A beautiful dark theme with a floating, draggable panel.
- **Keyboard Shortcut:** Press the `ESC` key to stop picking mode.
- **Privacy First:** No data tracking, no analytics, everything stays in your browser.

## 🖥️ Installation

Since this extension isn't on the Chrome Web Store yet, you can install it manually in developer mode.

1.  **Download the Code:**
    *   On this page, click the green **"Code"** button, then select **"Download ZIP"**.
    *   Extract the downloaded ZIP file to a folder on your computer.

2.  **Load the Extension in Chrome:**
    *   Open Google Chrome and go to `chrome://extensions/`.
    *   Turn on **"Developer mode"** using the toggle in the top right corner.
    *   Click the **"Load unpacked"** button.
    *   Select the folder where you extracted the ZIP file.
    *   The extension icon should now appear in your browser toolbar!

## 🚀 How to Use

1.  Click the extension icon in your browser toolbar to open the floating panel.
2.  Click the **"Start Picking"** button.
3.  Hover over any element on the webpage to highlight it.
4.  Click the element to inspect it.
5.  The panel will display the element's **Tag**, **ID**, **Classes**, and a detailed **CSS Selector**.
6.  To copy information, simply click on the **ID**, **Classes**, or **Selector** box.
7.  Press the `ESC` key on your keyboard to stop the picking mode.

## 🛠️ Technologies Used

*   **Vanilla JavaScript:** No external frameworks for the core logic.
*   **Font Awesome:** For clean and professional icons.
*   **Exo 2 Font:** A modern font from Google Fonts.

## 🔒 Privacy Policy

This extension **does not collect or transmit any data**. All operations are performed locally in your browser.

*   No tracking scripts
*   No analytics
*   No data is sent to any server (except for loading Font Awesome from its CDN).
